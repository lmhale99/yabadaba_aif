============
yabadaba_aif
============

Introduction
------------

This provides yabadaba extensions to support Adsorption Information File (AIF) records.

Features

- Build new AIF entries
- Load AIF files using Python and interact with the data
- Convert to equivalent JSON or XML representations
- Store and retrieve AIF records in a variety of database types.

Usage
-----

See the included Jupyter Notebooks for implementation and usage information.
